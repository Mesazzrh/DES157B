Coding Components & Feasibility Report

Features Plan to Include:

  • Desk based interface
    • Interactive objects users can click on
    • Hover effect for engagement
  • Branching decision paths
    • Each object leads to a small scenario with a design related decision
    • Users choose one of two options - one leads to utopia, the other dystopia
  • Future previews
    • After each decision, the user sees a mini-scene of a future world - stylized illustrations
  • Tracking outcome
    • Tracks which decisions the user makes
    • At the end, the system shows:
      • A percentage split (60% dystopian, 40% utopian)
      • A reflection summary: "You chose efficiency over creativity 2 out of 3 times"

Technologies Needed:

  • Softwares for designing visuals (Illustrator, Figma, etc.)
  • HTML, CSS, JavaScript
  • JS and CSS libraries

Feasibility:
